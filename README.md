# SMC Club Manager

Member management web app for the SMC club.

## Features
- Member tracking with status and dues management
- Speaker schedule management (synced with Google Sheet)
- Email composition with custom recipient selection
- Payment/dues tracking with anniversary-based billing
- Login authentication

## Tech Stack
- **Next.js 14** (React)
- **Vercel** (hosting)
- Future: Supabase (database), Resend (email), Stripe (payments)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connected to Vercel via GitHub. Push to `main` to auto-deploy.
